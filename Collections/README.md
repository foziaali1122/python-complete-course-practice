# Python Collections (List, Tuple, Dictionary, Set)

## Overview
This notebook contains a complete hands-on practice of Python core data structures including:

- Lists
- Tuples
- Dictionaries
- Sets

It demonstrates their properties, built-in methods, and real coding examples using Google Colab.

---

## Topics Covered

### 1. Lists
A list is an ordered, mutable, and indexed collection that allows duplicate values.

#### Features:
- Ordered
- Mutable (changeable)
- Indexed
- Allows duplicates

#### Methods Practiced:
- append()
- insert()
- remove()
- pop()
- extend()
- count()
- index()
- sort()
- reverse()
- copy()
- clear()

---

### 2. Tuples
A tuple is an ordered but immutable collection in Python.

#### Features:
- Ordered
- Immutable
- Allows duplicates

#### Concepts Covered:
- Tuple creation
- Indexing & slicing
- Counting elements
- Tuple unpacking
- Immutability handling (error demonstration)

---

### 3. Dictionaries
A dictionary is an unordered collection of key-value pairs.

#### Features:
- Key-value structure
- Keys are unique
- Mutable
- Fast lookup

#### Methods Practiced:
- get()
- keys()
- values()
- items()
- update()

#### Concepts Covered:
- Adding and updating values
- Safe key access
- Handling missing keys

---

### 4. Sets
A set is an unordered collection of unique elements.

#### Features:
- Unordered
- No duplicates allowed
- Unindexed

#### Methods Practiced:
- add()
- remove()
- pop()
- clear()

#### Set Operations:
- union()
- intersection()
- difference()
- symmetric_difference()
