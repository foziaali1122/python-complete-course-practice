# Python Control Flow (Conditions, Loops, Functions)

## Overview
This notebook provides a comprehensive hands-on practice of Python control flow concepts. It covers decision-making, looping, and function-based programming with real-world examples and beginner-friendly tasks.

---

## Topics Covered

### 1. Conditions (if, elif, else)
Conditions are used for decision-making in programs.

#### Concepts:
- if statement
- if-else statement
- if-elif-else ladder
- Nested conditions
- Logical operators (and, or, not)

#### Practice Tasks:
- Positive, negative, zero check
- Even/odd number
- Pass/fail system
- Voting eligibility
- Scholarship system
- Grade calculation
- Largest of three numbers
- Calculator using conditions
- Vowel/consonant check

---

### 2. Loops (for, while)
Loops are used to repeat tasks efficiently.

#### Types:
- for loop (used for iteration)
- while loop (runs until condition is true)

#### Concepts:
- Iteration over string, list, tuple, dictionary, set
- Range function
- Nested loops
- Loop with else

#### Practice Tasks:
- Print numbers (1 to N)
- Even and odd numbers
- Multiplication tables
- Reverse counting
- Sum and count of numbers
- Factorial calculation
- Pattern printing
- Palindrome check
- FizzBuzz problem

---

### 3. Loop Control Statements
Used to control loop execution.

- break (terminate loop)
- continue (skip iteration)
- pass (placeholder)

---

### 4. Functions
Functions are reusable blocks of code.

#### Concepts:
- Function definition and calling
- Parameters and arguments
- Return values
- Recursion
- Local and global variables
- LEGB rule (scope resolution)

#### Types of Arguments:
- Default arguments
- Keyword arguments
- Variable-length arguments (*args)

#### Practice Tasks:
- Sum of numbers
- Even/odd checker
- Maximum number finder
- Table generator
- Square and cube calculation
- Factorial using recursion
- Palindrome function
- Grade system
- Vowel counting
- List sum

---

### 5. Advanced Concepts

#### Variable Scope:
- Local variables
- Global variables

#### LEGB Rule:
- Local
- Enclosing
- Global
- Built-in

#### Recursion:
Function calling itself to solve problems (e.g., factorial, patterns)
