# Python Higher-Order Functions

## Overview
This notebook introduces the concept of **Higher-Order Functions** in Python. These are functions that either take other functions as input or return functions as output. They are widely used to write clean, efficient, and concise code.

---

## Key Concepts

### map()
Used to transform data by applying a function to each element in an iterable.

### filter()
Used to select specific elements from an iterable based on a condition.

### reduce()
Used to combine all elements of an iterable into a single result (such as sum or product).

### zip()
Used to combine multiple iterables into pairs (tuples).

### enumerate()
Used to add indexing while iterating over an iterable.
